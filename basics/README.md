# Shell, basics

Este proyecto marca el inicio en el manejo de la línea de comandos en sistemas Linux. Se cubren comandos fundamentales para la navegación, manipulación de archivos y directorios.

## Tareas

| Archivo | Descripción |
| --- | --- |
| 0-current_working_directory | Imprime la ruta absoluta del directorio actual. |
| 1-listit | Lista el contenido del directorio actual. |
| 2-bring_me_home | Cambia el directorio de trabajo al directorio home del usuario. |
| 3-listfiles | Lista el contenido del directorio actual en formato largo. |
| 4-listmorefiles | Lista el contenido incluyendo archivos ocultos. |
